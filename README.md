# Project 1: University Portal Learning Dashboard

## Description

A tablet-based university learning dashboard designed for first-time students. The dashboard provides an onboarding experience, course discovery, learning progress tracking, and academic management in a clean and intuitive interface.

## Key Features

* Tablet-first design
* Side navigation bar
* Student onboarding workflow
* Course exploration section
* Academic statistics overview
* Getting Started checklist
* Clean and minimal UI

# Project 2: SaaS Subscription Plan Matrix Selector

## Description

A modern SaaS pricing dashboard designed to help users compare subscription plans and select the most suitable package based on features, pricing, and business requirements.

## Key Features

* SaaS pricing comparison
* Monthly and annual billing options
* Feature comparison table
* Responsive layout structure
* Modern card-based UI
* Clear call-to-action buttons
